# 使い方

1. ターミナルを起動する
    1. カレントディレクトリを移動する。index.htmlがあるパスへ。`cd JS.Ruby.Parser.20220126113211/src`
    2. `python3 -m http.server 8000`
2. ブラウザを起動する
    1. `http://0.0.0.0:8000/` をURL欄に入力する
